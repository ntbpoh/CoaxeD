CoaxeD - version 0.0 

Noobie fumblings with the D language and GtkD

Initial foray into coaxing software to work in both Windows and Linux using the D language and the Gtk+3 (GtkD) framework. 
Most of this source has been pinched from existing examples that are available elsewhere. 
The project is only useful as a kind of scratchpad for developing working code to be used in other projects.
